Afin de bien comprendre comment se déroule la création d'une intelligence artificielle, nous allons vous guider avec une très courte introduction sur les fondamentaux de l'application.

Un modèle ?

L'objectif de cette application est de concevoir votre premier modèle d'IA. Un modèle est le "cerveau" d'une intelligence artificelle, avec lequel vous allez pouvoir faire des prédictions sur des données.

Configuration

La première étape consiste à configurer votre modèle. Cette étape permet de définir l'objectif de votre IA et son fonctionnement (il existe différents types de modèles).

Etiquetage

Comme pour un enfant, les modèles d'IA doivent apprendre (à reconnaitre un objet, par exemple). Si vous voulez que votre I.A. reconnaissent les chiens des chats il faudra étiqueter des images de chiens et de chats pour que l'IA puisse apprendre à les différencier.

Entraînement

Une fois que votre modèle est configuré et que vous avez étiqueté vos données, il est temps de l'entraîner. L'entraînement consiste à faire apprendre à votre modèle à reconnaître les données que vous lui avez fourni.

Prédiction

Une fois votre modèle entraîné, vous pourrez directement lui demander de faire des prédictions sur des données que vous lui fournirez. Vous pourrez aussi télécharger votre modèle pour l'utiliser dans d'autres applications !

Prêt ?

Vous êtes maintenant prêt à commencer à créer votre premier modèle d'IA !

Vous pourrez revenir à tout moment sur cette page en cliquant sur le bouton "Comment ça marche ?" dans le menu latéral.
